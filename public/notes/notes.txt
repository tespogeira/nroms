###Create p[roject]
> composer create-project laravel/laravel nroms

###Auth componente with Bootstrap
> composer require laravel/ui
> php artisan ui bootstrap
> npm install
> npm run dev (x2)

###Start Web Server
> php artisan serve

###List Routes
> php artisan route:list
